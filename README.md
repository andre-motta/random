# Welcome to random
  
The purpose of this repo is to hold all small random projects that I dedicate myself to throughout my journey in computer science. Here you will find all sorts of things:
  * From game solving using statistics
  * To production level snippets for commercial applications
  
### Most recent submission:
* [Wordle](https://github.com/andre-motta/random/blob/master/src/games/wordle) as of 2/11/2022

To find out more you can navigate around the repository! Start [here](https://github.com/andre-motta/random/blob/master/src) and have fun!


# Interacting with the repository

This repository is made public but is not open-source. Being managed by a single developer. [Me!](https://github.com/andre-motta)

## Suggestions

If you have a suggestion for a small problem please create a problem suggestion [here](https://github.com/andre-motta/random/issues/new/choose)

## Bug Reports

If you find any bugs or errors in the programs made available here please raise an issue [here](https://github.com/andre-motta/random/issues/new/choose)
